# MB Zolfaghari — Personal Site (Next.js 14 + contentlayer2)

Minimal‑luxe personal website with Resume, Articles (MDX), Books (MDX), and Videos (MDX).
Uses **contentlayer2/next-contentlayer2** (Next 14 compatible).

## Quick start
```bash
npm i
npm run dev
# or
pnpm i
pnpm dev
```

## Build
```bash
npm run build && npm start
```

## Content
- MDX lives in `content/` (articles, books, videos).
- Resume PDF at `public/resume.pdf`.
