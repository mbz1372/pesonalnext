# MB Zolfaghari — Personal Site (Next.js 14)

Minimal‑luxe personal website with Resume, Articles (MDX), Books (MDX), and Videos (MDX).

## Quick start
```bash
pnpm install
pnpm dev
```

## Build
```bash
pnpm build && pnpm start
```

## Content
- MDX lives in `content/` (articles, books, videos).
- Resume PDF at `public/resume.pdf`.
